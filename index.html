指针的作用：

1.在函数间传递数据。被调函数通过指针变量和取值操作直接访问主调函数中实参那里的数据（传址调用）(传值调用
实参的副本单向传给形参
被调函数访问的不是主调函数的数据区)

实参是地址，形参是指针变量。

可以提高函数见大量数据（比如数组）的传递效率，或改变主调函数中的数据（利用函数的副作用）

const int *p可以随便指，但是不能改

int * const p 声明时需要初始化，只能指一个，即初始化的那个，但可以改动地址

const int * const p 声明时需要初始化，只能指一个，即初始化的那个，也不能改动地址（啥都不能动）

2.访问动态变量。

动态变量和静态变量的区别:

动态变量: 1使用malloc 函数，在堆区获得空间，最后通过free释放空间

                 2自动变量，最普通的那类

要记得撤销动态变量，否则可能导致内存泄露/悬浮指针。free(pda);pda=NULL;

内存泄露：指针指的东西变了，原本被指的那些空间就泄露了

悬浮指针：例如在C/C++中，当通过 delete （C++）或者 free （C）操作释放了一块内存后，如果没有将指针赋值为 NULL ，这个指针就会变成悬浮指针。
